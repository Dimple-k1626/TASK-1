# Beginner Server App (Form Handling with Node.js)

## 📌 Project Description
This project is a simple **Node.js server application** built using **Express.js** and **EJS**.  
It displays a user registration form and processes the submitted data on the server.

When the user submits the form, the server receives the data and displays it on a result page.

---

## 🚀 Technologies Used
- Node.js
- Express.js
- EJS (Embedded JavaScript Templates)
- HTML
- Body-parser
